University of SUBWAY® Detection Test

Operating System Required
Microsoft Windows XP SP2 or higher, Apple Mac OS X v10.5 or higher, Red Hat Linux v3 or higher.
Browser Required
Internet Explorer v6.0 or higher, Google Chrome v10 or higher, Mozilla Firefox v3.0 or higher, or Apple Safari v3.0 or higher.
Adobe Flash Required
Adobe Flash plugin v9.0 or higher to be installed.
Java (JRE) Required
Java (JRE) plugin v1.5 or higher to be installed.
Popup Blocker Required
Blocker must be disabled, or configured to allow popups for: http://subway.sabanow.net.
CrystalReports Recommended
ActiveX and Java to be installed.

Popup Blocker Configuration

Please use the Official Documentation, on the bottom-left, to visit your browser's support page.

You may need to search for instructions on allowing popups for a specific website. Once you have made the necessary changes, try running the test again.


Some browsers require timed consistancy in which users must preform the action when prompted.

Installing the Add-On

  • Press the Install button on the bottom-right of this screen
  • Wait for the installation file to download to your system.
  • Your browser will then prompt you to authorize the installation, press Install on that dialog window.
  • Once complete, press 'Close' on the bottom-left, and re-run the test.
During the install, it may prompt for your account's password. This is to authenticate yourself as an administrator.
